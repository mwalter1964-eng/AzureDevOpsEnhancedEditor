DevO TinyMCE Editor - Privacy Policy
Last Updated: [Current Date]

Overview
The DevO TinyMCE Editor extension is an internal tool for editing Azure DevOps work items.

Data Collection
This extension does NOT collect, store, or transmit any user data.

What the Extension Does
Adds an edit button to Azure DevOps work item pages
Opens a rich text editor (TinyMCE) for editing work item content
Saves edited content back to Azure DevOps using Azure's native API
Data Processing
All data processing occurs locally in your browser
Work item content is read from and saved to Azure DevOps directly
No data is sent to external servers
No analytics or tracking is performed
No user information is collected
Permissions Used
activeTab: To detect Azure DevOps work item pages
scripting: To inject the edit button into the page
host_permissions (dev.azure.com ): To read and save work item content via Azure DevOps REST API
Third-Party Services
TinyMCE: A client-side text editor library (bundled with the extension, no external connections)
Azure DevOps: Work item data is read from and saved to your Azure DevOps organization
Contact
For questions about this extension, contact your development team.
